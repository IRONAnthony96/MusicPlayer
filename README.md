# MusicPlayer
Python音乐播放器

使用到的库有：
1. [pygame](https://www.pygame.org/docs/)
2. [tkinter](https://docs.python.org/3.6/library/tkinter.html)
3. [mutagen](https://mutagen.readthedocs.io/en/latest/)

## 版本0
实现的功能：
- 显示当前播放的歌曲名
- 播放上一首歌曲
- 播放下一首歌曲
- 停止播放
